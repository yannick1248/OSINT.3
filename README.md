# OSINT.3
Service de l’information mondiale.

## Vision produit unifiée

Cette base évolue vers un service OSINT unique avec trois modes d’usage sur le même moteur:

- **Web App**: exploitation terrain rapide, interface claire et offline-first.
- **IDE Analyste**: investigation avancée, scripts, validation des hypothèses.
- **CRM Renseignement**: gestion des dossiers, timeline et coordination équipe.

## Priorités techniques

1. Fusionner les flux (personnes, emails, téléphones, domaines, localisation, breaches) dans un graphe corrélé.
2. Garder une architecture modulaire pour activer/désactiver les modes Web App / IDE / CRM.
3. Préparer le socle offline et l’audit trail pour un usage opérationnel robuste.
